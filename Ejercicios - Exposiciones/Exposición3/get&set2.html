aqui creamos un mini proyecto de una lista de tareas (to-do list) en el que podremos agregar tareas, marcarlas como completadas y eliminarlas. Utilizaremos JavaScript para manipular el DOM y gestionar las tareas.

Lista de Tareas

    En este ejemplo, hemos creado una lista de tareas interactiva. Al ingresar una tarea en el campo de entrada y hacer clic en el botón "Agregar Tarea", se agrega la tarea a la lista. Cada tarea se puede marcar como completada al hacer clic en ella y se puede eliminar haciendo clic en un botón asociado. El código JavaScript utiliza tanto get como set, por ejemplo, al obtener referencias a elementos del DOM (getElementById) y al establecer atributos o contenido de elementos (textContent, classList, etc.).
